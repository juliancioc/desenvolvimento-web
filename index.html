Desenvolvimento
Web

Desenvolvimento web é o processo de criação e manutenção de sites
e aplicativos para a internet. Envolve várias disciplinas e habilidades.

Principais habilidades do desenvolvimento web

Front-end

Parte visual e interativa do site que os usuários veem e interagem. Tecnologias comuns incluem HTML, CSS e JavaScript.

Back-end

Parte do site que lida com a lógica, banco de dados e autenticação. Envolve servidores, bancos de dados e linguagens como Python, Ruby, PHP e Node.js.

Full-stack

Combinação de desenvolvimento front-end e back-end, onde o desenvolvedor lida com ambas as áreas.

Design Responsivo

Criação de sites que funcionam bem em diversos dispositivos e tamanhos de tela.
